⚡ Quick Guide
💰 Player ne deposit kiya: Players Tab → Search karo → 💰 Balance → Add → Amount enter → Save! Balance update ho jaega.

💸 Withdrawal approve karo: Withdrawals Tab → Request dekho → ✅ Approve → Player ko WhatsApp message jaega automatically.
🔍 Player dhundna: Players Tab → Search box mein naam, UID ya WhatsApp likho.
