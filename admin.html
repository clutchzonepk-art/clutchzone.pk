⚡ How to Use Admin Panel
💰 Player Deposit kiya: Players Tab → Search player by UID ya name → "💰 Balance" button → "Add to Balance" select karo → Amount enter karo → Save! Balance update ho jaega.
🏆 Tournament Banao: Tournaments Tab → "➕ Create New" button → Name, Time, Entry Fee, Prizes fill karo → Save! Tournament website pe live ho jaega.
💸 Withdrawal Approve: Withdrawals Tab → Request dekho → "✅ Approve" karo → Player ko automatically WhatsApp message jaega. Phir manually JazzCash/EasyPaisa se payment bhejo.
👥 Player Dhundna: Players Tab → Search box mein name, UID, ya WhatsApp number likho → Player mil jaega.
📱 Player se Contact: Players list mein "📱 WA" button dabao — seedha WhatsApp khul jaega us player ka.
